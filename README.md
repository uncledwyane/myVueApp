### 😃学习vue之后的一个小项目，集合多种功能的小项目。
功能：
+ ✔新闻
+ ✔号码归属地
+ ✔天气查询
+ ✔车型大全
+ ✔老黄历
+ ✔历史上的今天
+ ❌商城

技术栈：
+ Vue
+ Axios
+ VueRouter
+ Vant-UI

运行：
```shell
npm intall
```

打包：
```
npm run build
```

运行：
```
npm run dev
```
